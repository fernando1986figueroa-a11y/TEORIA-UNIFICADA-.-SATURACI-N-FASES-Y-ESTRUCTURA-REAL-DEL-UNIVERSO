# Contributing

Gracias por tu interés en revisar o ayudar a mejorar este repositorio. Algunas pautas rápidas:

- Issues: abre issues para discusiones, correcciones o solicitudes de formatos accesibles.
- Código: usa ramas descriptivas; añade tests y documentación. Mantén compatibilidad con Python 3.8+.
- Estilo: docstrings claros, comentarios en español o inglés, y small, focused commits.
- Revisión: se agradecen referencias y comprobaciones con la literatura.

Si necesitas versiones accesibles de los PDFs (texto plano, HTML, fuentes grandes), por favor crea un issue marcado `accessibility`.
